# Tres en Raya - Proyecto de Estructuras de Datos

## 1. Descripción
Juego de 3 en raya para dos jugadores (PvP) con historial de partidas.  
Implementa estructuras de datos lineales y no lineales: listas, pilas y grafos

## 2. Dependencias
- Java 8 o superior
- IDE: Eclipse (u otro compatible con Java)
- No requiere base de datos ni librerías externas

## 3. Cómo compilar y ejecutar
1. Abrir Eclipse y cargar el proyecto
2. Asegurarse de que todas las clases estén en el paquete `juego` dentro de `src`
3. Hacer clic derecho sobre `Main.java` 
4. Aparecerá el menú inicial con las opciones:
   - Empezar Juego: inicia una partida PvP
   - Ver Historial: muestra partidas jugadas, permite eliminar la ultima

## 4. Ejemplos de uso
- Empezar partida: Jugador 1 coloca “X”, luego Jugador 2 coloca “O” y así sucesivamente  
- Ganador: aparece un mensaje “Ganó X” o “Ganó O”  
- Empate: aparece un mensaje “Empate”  
- Historial: muestra todas las partidas jugadas en orden y permite eliminar la última  

## 5. Funcionalidades
- Juego PvP con tablero 3x3  
- Historial de partidas usando pilas
- Movimientos guardados en listas (ArrayList)  
- Contador de movimientos usando recursividad  
- Manejo de errores con try-catch para todas las acciones del juego
